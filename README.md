Running the application:
Download the Complete Library package from https://p5js.org/download/ into the /libraries folder.
Uncompress the p5.zip package.
Examine the p5 folder and move the p5.min and p5.sound.min files directly into the /libraries folder.
Change working directory to p5hadak folder.
Run one of these commands:
    python -m http.server
    python3 -m http.server
In browser, write localhost:8000/index.html.
